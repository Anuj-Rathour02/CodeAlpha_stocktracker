# CodeAlpha_stocktracker
A simple Python-based Stock Portfolio Tracker that allows users to add stocks, enter quantities, calculate investment values, and generate a portfolio summary. The program also saves the portfolio report to a text file.

# Features
Add multiple stocks to a portfolio
Calculate investment value for each stock
Display a complete portfolio summary
Calculate total investment value
Save portfolio report to a text file
Handle invalid stock symbols and inputs

# Technologies Used
Python

# Concepts Used
Dictionaries
Loops
Conditional Statements
Exception Handling
File Handling
Basic Arithmetic
